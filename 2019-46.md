# 云原生技术追踪 2019年第四十六期
## 分布式系统
**1.** **基于Raft构建大型分布式存储系统**

https://mp.weixin.qq.com/s/GT9JcwvtwPHTAXzWZjsM2Q

https://mp.weixin.qq.com/s/qfN5A76LCgxeohPwelqxVw

**摘要:** 基于 Hadoop 生态的大数据增量处理框架, 实现流批计算模式的统一.
